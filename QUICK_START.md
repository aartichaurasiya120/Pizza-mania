## 🎯 3 Steps to Start

### Step 1: Start Backend
```bash
cd backend
npm start
```

### Step 2: Start Frontend
```bash
cd Login
npm run dev
```

### Step 3: Access Admin
```
http://localhost:5173/admin/login
```
